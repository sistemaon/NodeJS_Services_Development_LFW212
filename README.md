## Install NodeJS
The recommended way to install Node.js on macOS and Linux is by using a Node version manager.\
Using **[NVM](https://github.com/nvm-sh/nvm)**.

##### Node Check:
```
node -v
```
##### NPM Check:
```
npm -v
```
